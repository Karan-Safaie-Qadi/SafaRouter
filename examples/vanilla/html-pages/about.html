درباره صفا-router

صفا-router یک مسیریاب前端 مستقل و حرفه‌ای است که از Next.js App Router الهام گرفته شده است.

چرا صفا-router؟

اگر الگوی Next.js App Router را دوست دارید اما نمی‌توانید از Next.js استفاده کنید — چون روی Create React App, Vite, یا یک پروژه HTML ساده کار می‌کنید — صفا-router همان قابلیت‌ها را بدون وابستگی به فریم‌ورک خاصی ارائه می‌دهد.

مفاهیم کلیدی

  • لایه‌بندی (Layouts) — صفحات را می‌پیچد و در طول مسیریابی پایدار می‌ماند
  • مسیرهای پویا — [slug] برای URLهای پارامتری
  • مسیرهای همه‌گیر — [...path] برای تطبیق انعطاف‌پذیر
  • میان‌افزار — هر مسیریابی را می‌توان intercept کرد
  • صفحات 404 / خطا — بازگشت داخلی با امکان سفارشی‌سازی
