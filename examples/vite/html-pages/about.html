درباره صفا-router (Vite)

صفا-router با Vite به راحتی کار می‌کند. پلاگین Vite به صورت خودکار import ویژگی‌های مورد نیاز را بر اساس کد شما تشخیص می‌دهد.

مزایای استفاده با Vite

  • HMR (Hot Module Replacement) در زمان توسعه
  • بهینه‌سازی build با Tree Shaking
  • تشخیص خودکار ویژگی‌ها و تزریق import
  • باندل بسیار کم حجم
