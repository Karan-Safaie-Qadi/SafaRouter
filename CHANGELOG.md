# Changelog

## 1.0.0 (2026-06-23)

- Initial release
- Route tree with nested layouts
- Dynamic segments `[param]`, `[...catchAll]`, `[[...optionalCatchAll]]`
- Route groups `(group)`
- Client-side navigation (History API)
- Middleware pipeline
- Custom 404 and error pages
- Event system
- Link component with active state
- Hash-based routing mode
- Base path support
- Route caching
- Zero dependencies, pure JavaScript
- Demo application with dark theme
