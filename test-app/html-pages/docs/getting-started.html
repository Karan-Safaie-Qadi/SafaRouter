Getting Started

Install SafaRouter and create your first route in under a minute.

npm install safa-router

Create a pages/ folder with index.html — that's your home page!

Basic HTML-only setup

const router = new SafaRouter({
  target: '#app',
  pageDir: 'html-pages',
})

router.start()

All page content is served from .html files — no JS exports needed.
