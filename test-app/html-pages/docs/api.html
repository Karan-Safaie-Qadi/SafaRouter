API Reference

Full documentation of the SafaRouter API.

Options

  • target — CSS selector or element
  • pageDir — Folder with .html files
  • layout — Global layout (function or HTML file)
  • routes — Explicit route definitions
  • components — Named components rendered on every page (e.g. header, footer)
  • basePath — Base URL path for all routes
  • useHash — Hash-based routing mode
