Vue 3 Integration

Using SafaRouter with Vue 3 via CDN or build tool.

Minimal Vue 3 + SafaRouter Setup

SafaRouter replaces Vue Router entirely. Your Vue app provides a mount point, SafaRouter handles all routing logic and page rendering. 

<template>
  <div ref="appRef"></div>
</template>

<script setup>
import { ref, onMounted, onUnmounted } from 'vue'
import { SafaRouter } from 'safa-router'

const appRef = ref(null)
let router

onMounted(() => {
  router = new SafaRouter({
    target: appRef.value,
    pageDir: 'html-pages',
    routes: { '/': {}, '/about': {} },
  })
  router.start()
})

onUnmounted(() => router?.destroy())
</script>

How It Works with Vue

  1. A Vue component exposes a container element via ref
  2. onMounted creates a SafaRouter instance targeting that element
  3. HTML pages are loaded and rendered inside the Vue-controlled container
  4. onUnmounted calls router.destroy() to prevent memory leaks
  5. Vue templates can use data-safa-link for SPA navigation

Using Vue Components as SafaRouter Smart Components

Your Vue components can return HTML strings that SafaRouter treats as smart components: 

// vueHeader.js
export default function vueHeader({ path }) {
  return [
    '<header style="border-bottom: 1px solid var(--color-border); padding: 0.75rem 1.5rem;">',
    '  <nav style="display: flex; gap: 1rem;">',
    '    <a href="/" data-safa-link',
    `      style="color: ${path === '/' ? 'var(--color-accent)' : 'var(--color-text-muted)'}">`,
    '      Home',
    '    </a>',
    '    <a href="/about" data-safa-link',
    `      style="color: ${path === '/about' ? 'var(--color-accent)' : 'var(--color-text-muted)'}">`,
    '      About',
    '    </a>',
    '  </nav>',
    '</header>',
  ].join('\n')
}

// main.js
import { SafaRouter } from 'safa-router'
import vueHeader from './vueHeader.js'

const router = new SafaRouter({
  target: '#app',
  pageDir: 'html-pages',
  components: { header: vueHeader },
  routes: { '/': {}, '/about': {} },
})
router.start()

CDN Script (Vue 3)

For HTML-only projects, include Vue 3 via CDN: 

<!-- index.html -->
<script src="https://unpkg.com/vue@3/dist/vue.global.prod.js"></script>
<script type="module">
  import { SafaRouter } from 'safa-router'

  const { createApp, ref, onMounted, onUnmounted } = Vue

  createApp({
    setup() {
      const appRef = ref(null)
      let router

      onMounted(() => {
        router = new SafaRouter({
          target: appRef.value,
          pageDir: 'html-pages',
          routes: { '/': {}, '/about': {} },
        })
        router.start()
      })

      onUnmounted(() => router?.destroy())

      return { appRef }
    },
  }).mount('#app')
</script>
