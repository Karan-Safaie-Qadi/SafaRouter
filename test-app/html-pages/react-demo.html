React Integration

Using SafaRouter with React via CDN UMD bundles.

Minimal React + SafaRouter Setup

SafaRouter is framework-agnostic. It manages routing while React handles the UI. The router owns a DOM target and your React app renders into it. 

import { useEffect, useRef } from 'react'
import { SafaRouter } from 'safa-router'

export default function App() {
  const appRef = useRef(null)
  useEffect(() => {
    const router = new SafaRouter({
      target: appRef.current,
      pageDir: 'html-pages',
      routes: { '/': {}, '/about': {} },
    })
    router.start()
    return () => router.destroy()
  }, [])
  return <div ref={appRef} />
}

How It Works

  1. React creates a root component that renders a container <div>
  2. useEffect initialises a SafaRouter instance targeting that container
  3. The router loads HTML pages and injects them into the container — React stays unaware of routing
  4. On unmount, router.destroy() cleans up event listeners
  5. React components can use data-safa-link anchors for client-side navigation

With React Router-style Components via SafaRouter

You can combine React components with SafaRouter's smart component system: 

// ReactApp.jsx
import { SafaRouter } from 'safa-router'

function reactHeader({ path }) {
  return (
    '<header>' +
    '  <nav>' +
    '    <a href="/" data-safa-link' +
    `      class="${path === '/' ? 'active' : ''}">Home</a>` +
    '    <a href="/about" data-safa-link' +
    `      class="${path === '/about' ? 'active' : ''}">About</a>` +
    '  </nav>' +
    '</header>'
  )
}

const router = new SafaRouter({
  target: '#app',
  pageDir: 'html-pages',
  components: { header: reactHeader },
  routes: { '/': {}, '/about': {} },
})
router.start()

CDN Script (React UMD)

For HTML-only projects, include React via CDN: 

<!-- index.html -->
<script crossorigin src="https://unpkg.com/react@18/umd/react.production.min.js"></script>
<script crossorigin src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js"></script>
<script type="module">
  import { SafaRouter } from 'safa-router'

  const router = new SafaRouter({
    target: '#app',
    pageDir: 'html-pages',
    routes: { '/': {}, '/about': {} },
  })
  router.start()
</script>
