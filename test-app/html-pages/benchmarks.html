Performance Notes

  • Zero dependencies — no framework overhead, no transitive deps
  • Fast route matching — prefix-tree (trie) based matching, O(k) where k = path segment count
  • LRU caching — loaded pages are cached (configurable, default 50 entries)
  • No parsing step — routes are defined as plain objects, no JSX or template compilation needed
  • History API — leverages native pushState / popState for zero-overhead navigation
  • Minimal DOM writes — only the target container is updated on each navigation

Why Size Matters

Every KB of JavaScript affects page load time, especially on mobile devices. SafaRouter's <5 KB gzipped footprint means it downloads in a single network round-trip on most connections, parses in under 1 ms, and leaves more budget for your application code. 

# SafaRouter install
npm install safa-router   # +0 transitive deps

# React Router install
npm install react-router-dom   # + react, scheduler, etc. (~30 KB gzipped)

# Vue Router install
npm install vue-router   # + vue (~30 KB gzipped for Vue alone)
