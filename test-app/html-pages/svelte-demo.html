Svelte Integration

Using SafaRouter with Svelte 5 via CDN or build tool.

Minimal Svelte + SafaRouter Setup

SafaRouter replaces SvelteKit's routing layer entirely. Your Svelte app provides a mount target, SafaRouter handles all routing logic and HTML page rendering. 

<script>
  import { onMount, onDestroy } from 'svelte'
  import { SafaRouter } from 'safa-router'

  let appRef
  let router

  onMount(() => {
    router = new SafaRouter({
      target: appRef,
      pageDir: 'html-pages',
      routes: { '/': {}, '/about': {} },
    })
    router.start()
  })

  onDestroy(() => router?.destroy())
</script>

<div bind:this={appRef} />

How It Works with Svelte

  1. A Svelte component exposes a container element via bind:this
  2. onMount creates a SafaRouter instance targeting that element
  3. HTML pages are loaded and rendered inside the Svelte-controlled container
  4. onDestroy calls router.destroy() to prevent memory leaks
  5. Svelte templates can use data-safa-link anchors for SPA navigation

Using Svelte Components as SafaRouter Smart Components

Your Svelte functions can return HTML strings that SafaRouter treats as smart components: 

// svelteHeader.js
export default function svelteHeader({ path }) {
  return [
    '<header style="border-bottom: 1px solid var(--color-border); padding: 0.75rem 1.5rem;">',
    '  <nav style="display: flex; gap: 1rem;">',
    '    <a href="/" data-safa-link',
    `      style="color: ${path === '/' ? 'var(--color-accent)' : 'var(--color-text-muted)'}">`,
    '      Home',
    '    </a>',
    '    <a href="/about" data-safa-link',
    `      style="color: ${path === '/about' ? 'var(--color-accent)' : 'var(--color-text-muted)'}">`,
    '      About',
    '    </a>',
    '  </nav>',
    '</header>',
  ].join('\n')
}

// main.js
import { SafaRouter } from 'safa-router'
import svelteHeader from './svelteHeader.js'

const router = new SafaRouter({
  target: '#app',
  pageDir: 'html-pages',
  components: { header: svelteHeader },
  routes: { '/': {}, '/about': {} },
})
router.start()

CDN Script (Svelte 5)

For HTML-only projects, include Svelte 5 via CDN: 

<!-- index.html -->
<script src="https://unpkg.com/svelte@5/dist/svelte.js"></script>
<script type="module">
  import { SafaRouter } from 'safa-router'

  const { mount, unmount } = Svelte

  const app = mount(App, { target: document.getElementById('app') })
</script>
