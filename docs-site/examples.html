Integration Examples

SafaRouter works with any JavaScript framework. Here are minimal integration patterns.

React

import { useEffect, useRef } from 'react'
import { SafaRouter } from 'safa-router'

export default function App() {
  const appRef = useRef(null)

  useEffect(() => {
    const router = new SafaRouter({
      target: appRef.current,
      pageDir: 'html-pages',
      routes: {
        '/': {},
        '/about': {},
        '/blog': {
          children: { '[slug]': {} },
        },
      },
      components: {
        header: ({ path }) => `
          <nav>
            <a href="/" data-safa-link>Home</a>
            <a href="/about" data-safa-link>About</a>
          </nav>
        `,
      },
    })

    router.start()
    return () => router.destroy()
  }, [])

  return (
    <div>
      <div data-safa-component="header"></div>
      <main ref={appRef}></main>
    </div>
  )
}

import { ref, onMounted, onUnmounted } from 'vue'
import { SafaRouter } from 'safa-router'

const appRef = ref(null)
let router

onMounted(() => {
  router = new SafaRouter({
    target: appRef.value,
    pageDir: 'html-pages',
    routes: {
      '/': {},
      '/about': {},
      '/user/[id]': {},
    },
    components: {
      header: ({ path }) => `
        <nav>
          <a href="/" data-safa-link>Home</a>
          <a href="/about" data-safa-link>About</a>
        </nav>
      `,
    },
  })
  router.start()
})

onUnmounted(() => router?.destroy())
</script>

Svelte

<div data-safa-component="header"></div>

Complete Demo Configuration

Here is a full-featured example showing all the capabilities together:

import { SafaRouter } from 'safa-router'
import headerComponent from './Header.js'
import footerComponent from './Footer.js'

const router = new SafaRouter({
  target: '#app',
  basePath: '/my-app',
  pageDir: 'html-pages',

  layout: 'html-pages/_layout.html',

  components: {
    header: headerComponent,
    footer: footerComponent,
  },

  titleTemplate: '%s — My App',

  errors: {
    pageDir: 'html-pages/errors',
    stackTraces: true,
  },

  access: {
    blocked: ['/admin', '/private/**'],
    ignored: ['/hidden'],
  },

  maintenanceMode: {
    enabled: false,
    allowedPaths: ['/login', '/assets/**'],
  },

  routes: {
    '/': {},

    '/blog': {
      children: { '[slug]': {} },
    },

    '/dashboard': {
      layout: ({ children }) => `
        <aside>Sidebar</aside>
        <div class="content">${children}</div>
      `,
      children: { settings: {} },
    },

    '/slow': {
      loading: () => '<div class="spinner">Loading…</div>',
      loader: async () => {
        await new Promise(r => setTimeout(r, 1500))
        return { data: 'loaded' }
      },
    },

    '/guard': {
      guard: ({ router }) => {
        const authed = localStorage.getItem('auth') === 'true'
        return authed || '/login'
      },
    },

    '/transition': {
      meta: {
        transition: {
          duration: 500,
          enterClass: 'fade-in',
          enterActiveClass: 'fade-in-active',
        },
      },
    },
  },
})

router.afterEach(({ pathname }) => {
  console.log(`Navigated to: ${pathname}`)
})

router.onError(({ path, error, statusCode }) => {
  console.warn(`Error at ${path}:`, error.message)
})

router.start()

TypeScript Example

import { defineRoute, defineRoutes, createRouter } from 'safa-router/typed'

const routes = defineRoutes({
  '/': {},
  '/blog/[slug]': {
    loader: async ({ params }) => {
      // params.slug is inferred as string
      const data = await fetch(`/api/posts/${params.slug}`)
      return data.json()
    },
  },
  '/docs/[...path]': {
    loader: async ({ params }) => {
      // params.path is inferred as string[]
      return { segments: params.path }
    },
  },
})

createRouter({
  target: '#app',
  routes,
}).start()

Vite Plugin

// vite.config.js
import { safaRouter } from 'safa-router/vite'

export default {
  plugins: [safaRouter()],
}

The plugin auto-detects which optional features you use and injects only the necessary imports, keeping your production bundle minimal.

SafaDevServer

import { SafaDevServer } from 'safa-router'

const server = new SafaDevServer({
  port: 3000,
  root: './test-app',
  basePath: '/test-app',
  watch: true,
  watchDirs: ['./test-app/html-pages'],
})

server.start()
// http://localhost:3000/test-app/

Parallel Routes (Slots)

routes: {
  '/dashboard': {
    page: () => '<h1>Dashboard</h1>',
    slots: {
      sidebar: {
        loader: async () => {
          const res = await fetch('/api/sidebar')
          return res.json()
        },
      },
    },
  },
}
<main id="app"></main>
<aside data-safa-slot="sidebar"></aside>

Intercepting Routes

routes: {
  '/photo/[id]': {
    intercept: true,  // always show as overlay on internal nav
    page: ({ params }) => `
      <div class="modal">
        <img src="/photos/${params.id}.jpg" />
      </div>
    `,
  },
}

// Programmatically dismiss
router.dismissInterceptor()
💡

Framework-agnostic by design. SafaRouter doesn't require React, Vue, or Svelte — it works with any of them because all components are plain functions returning HTML strings. Choose the framework you love, or use none at all.
